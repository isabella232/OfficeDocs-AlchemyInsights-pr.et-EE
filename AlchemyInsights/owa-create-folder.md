# <a name="create-a-folder"></a>Kausta loomine

1. Valige kausta Outlook veebis kaustaspaani allosas **Uus kaust**. **Uue kausta** nupu nägemiseks peate võib-olla kaustaspaanile alla kerima.
1. Sisestage tekstiväljale kausta nimi ja vajutage sisestusklahvi (ENTER).

Lisateavet leiate jaotisest [töötamine sõnumikaustadega Outlookis veebis](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
