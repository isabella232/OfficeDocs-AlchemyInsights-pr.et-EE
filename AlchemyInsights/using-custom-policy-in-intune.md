# <a name="using-custom-policy-in-intune"></a>Kohandatud poliitika kasutamine Intune's

Intune'is on palju androidi, iOS-i, macOS-i ja Windows sätteid. Samuti saate luua kohandatud profiili ja seejärel juurutada need kohandatud sätted Intune'i abil oma seadmetesse. Kohandatud profiilid on loodud lisama seadmesätteid ja funktsioone, mis pole Intune'i sisse ehitatud.
