# <a name="workflows-in-sharepoint"></a>SharePointi töövood

Töövoogude SharePointi abil saate mudel ja automatiseerida äriprotsesse. SharePoint toetab kahe peamise töövoo arengu keskkondades authoring töövood: SharePoint Designer ja Visual Studio. 

Vaata [arendada SharePointi töövoogu SharePoint Designer ja Visio](https://docs.microsoft.com/en-us/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) või [SharePoint arendada töövoogude kasutades Visual Studio](https://docs.microsoft.com/en-us/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) hinnata plusse ja miinuseid iga. 

Kui töövood on lahendus kasutada oma SharePoint keskkond või on üldinfot alustamise kohta, ülevaate [SharePointi töövood](https://docs.microsoft.com/en-us/sharepoint/dev/general-development/get-started-with-workflows-in-sharepoint#overview-of-workflows-in-sharepoint).
