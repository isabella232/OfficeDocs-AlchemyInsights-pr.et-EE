# <a name="reset-the-users-password"></a>Kasutaja parooli lähtestamine

Peate lähtestama kasutaja parooli, enne kui teete midagi muud. Tehke järgmist.

1. **Kasutajate**  >  **[Aktiivsete kasutajate](https://go.microsoft.com/fwlink/p/?linkid=834822)** valimine.
1. Valige kasutaja, kelle konto on ohustatud.
1. Klõpsake kuvatavas hüpik nuppu **Lähtesta parool**.

> [!NOTE]
> Ära saada meili teel parooli kasutajale.
