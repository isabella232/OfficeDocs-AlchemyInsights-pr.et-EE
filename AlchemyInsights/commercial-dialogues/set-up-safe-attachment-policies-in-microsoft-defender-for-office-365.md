# <a name="set-up-safe-attachment-policies-in-microsoft-defender-for-office-365"></a>Turvaliste manuste poliitikate häälestamine rakenduses Microsoft Defender for Office 365

1. Avage [Office 365 turbe & nõuetele vastavuse keskus](https://go.microsoft.com/fwlink/p/?linkid=2077143) ja valige **ohustatud halduse**  >  **poliitika**  >  **ohutud manused**.
2. Selle funktsiooni lubamiseks muude töökoormuste korral valige **SharePoint Online ' i, OneDrive ' i ja Microsoft teamsi jaoks suvand Lülita Microsoft Defender for Office 365**.
3. Uue poliitika loomiseks valige **Uus** **+** ikoon.
4. Määrake poliitika nimi, kirjeldus ja sätted.
5. Valige **Salvesta**.
