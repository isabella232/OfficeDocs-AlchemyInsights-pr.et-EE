# <a name="cant-add-default-2010-approval-workflow"></a>Ei saa lisada vaikimisi 2010 kinnitamise töövoog

Microsoft SharePointi saidikogumi, ei saa lisada globaalselt korduvkasutatav töövoog (nt "kinnitamine-SharePoint 2010") loendisse või teeki. Kinnitamise töövoog võib puududa. Selle probleemi lahendamiseks vaadake [ei saa lisada globaalselt korduvkasutatava töövoo SharePointi loendi või teegi](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Lisateavet töövoo kinnituste kohta leiate jaotisest [Kõik kinnitamise töövoogude](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742)kohta. 
 
Vaadake lisateavet [moodsate kinnituste voo](https://flow.microsoft.com/blog/introducing-modern-approvals)kohta.
