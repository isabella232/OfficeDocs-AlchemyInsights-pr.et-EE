# <a name="turn-off-focused-inbox-in-email-clients"></a>Lülita suunatud sisendkausta e-posti kliendid

Suunatud sisendkausta välja lülitada Outlook 2016, Outlook veebis, Outlook.com/Hotmail.com, Outlook 2016 for Mac või Mail for Windows 10, vaadake [Lülita suunatud sisendkausta](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Et välja lülitada suunatud sisendkausta kõigile organisatsioonis, vaadake [konfigureerida suunatud sisendkausta kõigile oma organisatsioonis](https://docs.microsoft.com/office365/admin/setup/configure-focused-inbox).
