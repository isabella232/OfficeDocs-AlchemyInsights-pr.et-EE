# <a name="fix-problems-with-the-download-installation-and-update-of-microsoft-edge"></a>Failide allalaadimise, installimise ja värskendamisega seotud probleemide Microsoft Edge

Enne paranduste proovimist veenduge, et teie arvuti operatsioonisüsteem ühildub Microsoft Edge-ga, mis toetab Windows 10, 8, 8.1, 7 32-bitist (x86) ja 64-bitist (x64) ning macOS 10.12 Sierrat ja uuemat versiooni. Allalaadimise, installimise Microsoft Edge värskendamisega seotud probleemide lahendamiseks järgige järgmisi juhiseid.

1. Kontrollige võrku ja VPN-i.
    - Kontrollige võrguühendust.
    - Tõrge 403 viitab sellele, et VPN võib teie allalaaditud faile blokeerida. Katkestage ühendus VPN-iga ja proovige Microsoft Edge uuesti.
1. Lisage officeapps.live.com oma brauseri usaldusväärsete veebisaitide loendisse.
    Näiteks kui kasutate Internet Explorerit selle allalaadimiseks, Microsoft Edge.
    1. Valige **Tööriistad**  >  **Interneti-suvandid**.
    2. Valige vahekaardil **Turve** väärtus **Usaldusväärsed**  >  **saidid.**
    3. Tippige **jaotises Lisa see veebisait tsooni**, valige Lisa ja seejärel valige <https://officeapps.live.com> **Sule**. 
1. Installige Microsoft Edge arvutisse Windows, ei pea te seda enne uuesti Microsoft Edge desinstallima. Samuti ei mõjuta uuesti installimine teie ajalugu, küpsiseid ega sätteid.

    Mac-arvutis peate selle enne Microsoft Edge desinstallima. Samuti peate taastama oma ajaloo, küpsised ja sätted.

    Maci Microsoft Edge desinstallimiseks:
    1. Avage Finder.
    2. Valige **kaustas** Rakendused **Microsoft Edge**.
    3. Valige **Fail Teisalda**  >  **prügikasti.**

    Arvutisse Microsoft Edge või Mac-arvutisse Windows uuesti installimiseks:
    1. Avage mis tahes tööbrauser.
    2. Brauseri allalaadimiseks Microsoft Edge uuesti installimiseks avage Microsoft Edge allalaadimise veebisait.
1. Taaskäivitage arvuti. Taaskäivitage arvuti ja proovige uuesti Microsoft Edge.
