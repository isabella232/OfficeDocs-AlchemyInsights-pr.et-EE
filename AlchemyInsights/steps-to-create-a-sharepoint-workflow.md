# <a name="steps-to-create-a-sharepoint-workflow"></a>Samme, et luua SharePointi töövoo

- Kui te ei saa SharePoint 2010 kinnitamise töövoo lisamine, [ei saa lisada vaikimisi 2010 kinnitamise töövoo](https://docs.microsoft.com/alchemyinsights/can-t-add-default-2010-approval-workflow).
- Kuidas luua töövood:
    - [2013 töövoo SharePoint Designer](https://docs.microsoft.com/sharepoint/dev/general-development/creating-a-workflow-by-using-sharepoint-designer-and-the-sharepoint-wo)
    - [2010 töövoo SharePoint Designer](https://support.office.com/article/introduction-to-designing-and-customizing-workflows-32c9c0bf-5e20-4f74-8b9c-d3ea79f2962b)
    - [Sissejuhatus SharePointi töövoo](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3)

- Töövoo tõrkeotsingu ja silumise:
    - Silumine töövood meenutab eelmise versiooni.  Lisateabe saamiseks vt [SharePointi silumine töövood](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).
    - Tõrkeotsingu kohta lisateabe saamiseks vt [tõrkeotsing SharePoint Server töövoo valideerimisvead](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).
