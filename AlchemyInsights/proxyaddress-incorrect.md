# <a name="proxyaddress-incorrect"></a>ProxyAddress on vale

Kui objekt on sünkroonitud Azure AD, väärtused, mis on määratud Active Directory atribuuti proxyAddresses võrreldakse Azure AD reeglid ja seejärel atribuuti proxyAddresses asustatud Azure AD. Seetõttu Active Directory objekti atribuuti proxyAddresses väärtused ei pruugi olla sama atribuuti proxyAddresses Azure AD väärtused.
  
Lisateavet selle kohta, kuidas Kasutajaatribuut on asustatud, vaadake, [Kuidas atribuuti Kasutajaatribuut asustatud Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
