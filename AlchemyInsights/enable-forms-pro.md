# <a name="enable-or-disable-forms-pro"></a>Vormide Pro lubamine või keelamine

Kui proovite lubada Microsoft Forms Pro, see on kaasatud [Dynamics 365 Enterprise](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) litsentsi osana. Teise võimalusena saab administraator osta [Microsoft Forms Pro litsentsi](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant). Saate eemaldada ka kasutaja [Microsoft Forms Pro litsentsi](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) .
