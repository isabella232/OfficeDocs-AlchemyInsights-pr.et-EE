# <a name="one-of-your-on-premises-federation-service-certificates-is-expiring"></a>Üks teie asutusesisese Föderatsiooni tunnistuse lõpeb

Probleemi lahendamiseks toimige järgmiselt.
  
- Microsoft Azure Active Directory moodul jaoks Windows PowerShelli installimine arvutisse (kui moodul pole juba installitud). Selleks minge [Azure Active Directory PowerShelli graafik](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)
    
- Järgige selle "stsenaarium 1: AD FS luba sisselogimise sert aegunud" osa ["Ilmnes probleem saidile" tõrge, kui välise kasutaja logib sisse Office 365, Azure, või Intune ADFS](https://support.microsoft.com/help/2713898/there-was-a-problem-accessing-the-site-error-from-ad-fs-when-a-federat).
    
- Järgige t[Kuidas uuendada või remont ühendatud Domeen Office 365, Azure, või Intune seadeid](https://support.microsoft.com/help/2647048/how-to-update-or-repair-the-settings-of-a-federated-domain-in-office-3).
    
Föderatsiooni sertifikaatide uuendamise kohta lisateabe saamiseks vt [sertifikaadi uuendamise O365 ja Azure AD](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-o365-certs).
