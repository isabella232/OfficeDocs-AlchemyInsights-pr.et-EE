# <a name="cutover-migration"></a>Ülelõigemigratsioon

Kasutajate postkastide sisu migreerimine üleliidetud [migreerimise abil.](https://admin.microsoft.com/adminportal/home#/cutoverwizard) See meetod, mis toetab kuni 2000 postkasti teisaldamist, teisaldab meilikasutajad, meilikontaktid ja meilitoega rühmad liikmestaatusega. Head tavad dikteerivad migreerimist kuni 150 kasutajaga rühmades, kuna protsessi toiminguid tehakse.
