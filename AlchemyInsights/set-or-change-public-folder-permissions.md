# <a name="permissions-and-public-folders"></a>Õigused ja avalikud kaustad

Saate muuta oma ühiskaustade õigusi, kasutades Outlooki, Exchange ' i halduskeskus (EAC) või PowerShelli:
  
- Outlooki juhiste saamiseks [klõpsake siin](https://support.office.com/article/Set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005.aspx).
    
- EAC, vaadake [käesoleva artikli](https://technet.microsoft.com/library/jj651147%28v=exchg.150%29.aspx.aspx#Anchor_1) juhiseid. 
    
- PowerShelli, lugege [käesoleva artikli](https://technet.microsoft.com/library/bb124743%28v=exchg.160%29.aspx.aspx) juhiseid, kasutades lisa PublicFolderClientPermission commandlet. Kui teil on vaja juhiseid ühenduse loomiseks Exchange PowerShelli, klõpsake [siin](https://technet.microsoft.com/library/jj984289%28v=exchg.160%29.aspx.aspx).
    
Kui **Välised kasutajad ei saa saata e-kirju Meililoaga ühiskausta**, põhjus võib olla, et ühiskausta puudub õigused, mis on vajalikud välise meili kohaletoimetamiseks. Saate seda parandada, kasutades Outlooki juhiseid [siin](https://technet.microsoft.com/library/aa997560%28v=exchg.150%29.aspx.aspx#Anchor_1)või PowerShelli juhiseid [siin](https://support.microsoft.com/help/2984402/-5.7.1-smtp-550-5.7.1-resolver.rst.authrequired-nondelivery-report-when-external-users-try-to-send-mail-to-mail-enabled-public-folders-in-office-365.aspx).
