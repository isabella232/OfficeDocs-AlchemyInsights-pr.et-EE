# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>SharePointi failide sünkroonimine uue OneDrive’i sünkroonimisrakendusega

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Käsk Ava Exploreris avab Windows Exploreri kohaliku eksemplari, mis kuvab SharePointi saiti majutava serveri kausta struktuuri. Seetõttu soovitame <a href="https://support.office.com/et-EE/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">sünkroonida SharePointi failid uue OneDrive’i sünkroonimisrakendusega</a>, mis pakub <a href="https://support.office.com/et-EE/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">faile nõudmisel</a>, kuna see võimaldab failidele juurdepääsu ja pakub parimat jõudlust</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Kui otsustasite uue sünkroonimisrakenduse asemel kasutada uurija vaadet, veenduge, et järgiksite alltoodud artiklites toodud juhiseid ja häid tavasid.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/et-EE/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">SharePoint Online’i probleemide tõrkeotsinguks käsu Ava Exploreris kasutamine</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/et-EE/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">Käsu Ava Exploreris abil teegifailide kopeerimine või teisaldamine</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Märkus.</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> Uues teegi versioonis nuppu Ava Exploreris ei ole. Klõpsake üleval paremal rippmenüüd <strong>Vaade</strong> (rippmenüü nimi muutub olenevalt teie praegusest vaatest) ja seejärel klõpsake File Exploreris nuppu <strong>Vaade</strong>.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Märkus.</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> SharePointi käsk Ava Exploreris kasutab <strong>ActiveX-juhtelemente</strong>, mistõttu seda toetatakse ainult Internet Exploreri versioonis 10 või 11. Käsk Ava Exploreris ei tööta Windowsis Microsoft Edge’i, Google Chrome’i, Mozilla Firefoxi ega Maci platvormiga. Seepärast võib valik &ldquo;Exploreri vaade&rdquo; olla tuhmina kuvatud.</span></p> <ul> <li>
