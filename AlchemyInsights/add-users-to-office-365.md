# <a name="add-a-user"></a>Kasutaja lisamine

1. Avage Microsoft 365 halduskeskuses **Kasutajad** > [Aktiivsed kasutajad](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users).

2. Valige **Kasutaja lisamine**.

3. Kasutajateabe, tootelitsentside, valikuliste rollide ja profiiliteabe lisamiseks järgige viisardit.

Lisateabe saamiseks vaadake teemat [Kasutajate ükshaaval või mitmekaupa lisamine](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users).
