# <a name="setting-up-a-hybrid-environment"></a>Seadistamine hübriid keskkonnas

Lase sujuvalt koos töötada ükskõik kui nende andmed on salvestatud asutusesisestest serveritest või Office 365 kasutajatele, peate võib-olla luua hübriid keskkonnas. Saate vaadata oma võimalusi, Vaata [Office 365 integratsiooni kohapeal keskkondades](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
